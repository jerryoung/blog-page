1.Algorithm：每周至少做一个 leetcode 的算法题

解题思路：前缀和 + 哈希缓存

  1. sum{nums[x] + num[x + 1]…. nums[i]} == k
  2. sum - pre[x] == k
  3. sum - k == pre[x]
  4. 通过 sum - k，通过map寻找 pre[x]，如果没有则返回0，否则累加次数
  5. 缓存sum到map，如果map中不存在sum到话存储0 + 1，否则获取到原来到次数 + 1存储
